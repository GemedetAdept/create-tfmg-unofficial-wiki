# An Unofficial Wiki for Create: The Factory Must Grow

*Create: The Factory Must Grow* is an addon for the Minecraft **Create** mod created by **Dr_Mango_Tea**.

The wiki pages are currently being stored on the [Wiki](https://github.com/GemedetAdept/create-tfmg-unofficial-wiki/wiki) tab of the repo. I may end up creating a GitHub Pages site in the future so it looks a bit cleaner.
